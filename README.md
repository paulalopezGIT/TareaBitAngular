# BIT Angular

* [NodeJs](https://nodejs.org/es/)

`node -v`
* [Npm](https://www.npmjs.com/)

`npm -v`
* [Angular CLI](https://cli.angular.io/)

`ng version`

* [Git](https://git-scm.com/downloads)

`git `
